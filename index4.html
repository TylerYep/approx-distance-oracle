Instructions:

  • Hover over any node to see its bunch [grey lines] (the distances calculated for that node) and its witnesses [colored dashed circles] (the nodes that are closest to it in each randomly selected subset of the graph).
  • Click on a node to select it as a starting node for a query. This will fix its bunch and witnesses in place. To change starting node, simply click a different node.
  • Hover over any other node to see its respective bunch and witnesses, and the approximate distance path [black arrows] returned by the ADO and the true shortest path [red arrow]. 
  • Adjust the slider bars to change k (the number of subsets chosen in the graph) and n (the number of nodes in the graph). All nodes will be redrawn for each change.
